Q.1 What is Format Specifier in C programming language with example ?

Ans.

Format Specifier :

  1. The Format Specifier is a way to tell the compiler what type of data is in a variable during taking input and display output to the user
  2. We need to use Format Specifier whether we are printing formated output with printf() or accepting input with scanf() .
  3. It is a placeholder of variables.
  4. Each Format Specifier must begin with modoul ( % ) sign.

Some of the most common % specifier that we can use in ANSIC are as follows :

    %c : A character
    %d : A integer
    %f : Float
    %l : Long int
    %lf : Double
    %Lf : Long double

  3. It is composed of two or more character starting with backslash ( \ ) .

    4. There are four types of control statements in C :

  4. (i) Decision making statements

Q.4 What is Decision Making Statement or the if-else Statement in C ?

Ans.

Decision making statement :

  1. Decision Making Statements in programming languages decides the direction of flow of program execution
  2. In decision control statements (if-else and nested if), group of statements are executed when condition is true. If condition is false, then else part statements are executed.

    3. There are 3 types of Decision Making control statements in C language :

  3. i) if statements
    Syntax :
    if(condition)
    {
    // code / statements
    }

    ii) if else statements
    Syntax :
    if (condition)
    {
    // Executes this block
    // condition is true
    }
    else
    {
    // Executes this block if
    // condition is false
    }

    iii) nested if statements
    Syntax:
    if (condition1)
    {
    // Executes when condition1 is true
    if (condition2)
    {
    // Executes when condition2 is true
    }
    }

Q.5 What is Switch case statement in C ?

Ans.

Switch case statement :

  1. A Switch case statementallows a variable to be tested against several values for equality. Each of these values is called a case. Once a matching case is found, its particular block of code is executed
  2. The Switch case statement is a multiway branch statement. It provides an easy way to dispatch execution to different parts of code based on the value of the expression.
  3. Switch is a control statement that allows a value to change control of execution.
    4. Syntax :

    switch (n)
    {
    case 1: // code to be executed if n = 1;
    break;
    case 2: // code to be executed if n = 2;
    break;
    default: // code to be executed if n doesn't match any cases
    }

Q.7 What are loops in C programming language ?

Ans.

loops in C :

  1. A loop executes the sequence of statements many time until the stated condition becomes false . A loop consists of two parts, a body of a loop and a control statement.
  2. In computer programming, a loop is a sequence of instructions that is repeated until a certain condition is reached. An operation is done, such as getting an item of data and changing it, and then some condition is checked such as whether a counter has reached a prescribed number.

Q.8 Type of loops in C programming language ?

Ans.

Type of loops in C :

  1. While Loop :

    (I) Repeats a statement or group of statements while a given condition is true. It tests the condition before executing the loop body
    (II) While loop is mainly consisted of three statements – initialization expression, test expression, update expression. The syntax of the three loops – For, while and do while mainly differs on the placement of these three

  2. For Loop :

    (I) Executes a sequence of statements multiple times and abbreviates the code that manages the loop variable.
    (II)The initialization statement is executed only once. Then, the test expression is evaluated. If the test expression is evaluated to false, the for loop is terminated.
    (III)However, if the test expression is evaluated to true, statements inside the body of for loop are executed, and the update expression is updated. Again the test expression is evaluated.

  3. Do..While Loop :

    (I)Do..While Loop is more like a while statement, except that it tests the condition at the end of the loop body.
    (II)The basic format of while loop is as follows:
    do {
    statements
    }
    while (expression);

  4. Nested Loop :

    (I) We can use one or more loops inside any other while, for, or do..while loop.




Important :

Why use for loop :

1. The for loop is used to a part of the program several times.

2. It is used to traverse the data structures like the arrays and linked list.

3. It has a little different syntax than while and do while loops .

Syntax :

for(exp1;exp2;exp3)
{
code to be run
}
